## Descripción

**SAP Woo Suite Lite** es la versión gratuita del conector entre WooCommerce y SAP Business One. Permite sincronizar stock y precios desde SAP a tu tienda WooCommerce de forma automática.

### Características

- ✅ **Conexión a SAP Business One** via Service Layer (REST)
- ✅ **Sincronización de stock** desde almacén SAP configurado
- ✅ **Sincronización de precios** por tarifa seleccionada
- ✅ **Sincronización automática** via WP-Cron
- ✅ **Logs básicos** de sincronización
- ✅ **Panel de administración** intuitivo
- ✅ **Upgrade seamless** a versión PRO

### ¿Necesitas más?

Actualiza a [SAP Woo Suite PRO](https://replanta.net/sap-woo-suite-pro/) para:

- Importación completa de productos desde SAP
- Sincronización de pedidos a SAP
- Sincronización de clientes (Business Partners)
- Mapeo de campos personalizable
- Soporte multicanal (TikTok, Amazon, eBay)
- REST API para integraciones
- Soporte prioritario

## Instalación

### Desde WordPress.org

1. Ve a **Plugins → Añadir nuevo**
2. Busca "SAP Woo Suite Lite"
3. Haz clic en **Instalar ahora**
4. Activa el plugin

### Manual

1. Descarga el ZIP desde [GitHub Releases](https://github.com/replantadev/sap-woo-suite-lite/releases)
2. Ve a **Plugins → Añadir nuevo → Subir plugin**
3. Selecciona el archivo ZIP
4. Instala y activa

---

## Configuración

1. Ve a **SAP Woo Suite Lite** en el menú de WordPress
2. Configura la conexión a SAP:
   - URL del Service Layer
   - Base de datos SAP
   - Usuario y contraseña
3. Prueba la conexión
4. Configura almacén y tarifa para sincronización
5. Guarda los cambios
